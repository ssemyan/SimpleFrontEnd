Simple Front End

Simple HTML and Javascript front end for a Todo list. Set the URL of the local or backend API in vars.js You can also run without an API (items are added or deleted to the list but not persisted to any storage) by setting the noApi variable in vars.js to true.

An example WebApi backend can be found here: https://github.com/ssemyan/SimpleSqlBackend
